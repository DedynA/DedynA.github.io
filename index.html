My Portofolio

Awesome Work

Sistem Kontrol Ruang Menggunakan Aplikasi Telegram

Project ini merupakan tugas akhir (skripsi) kuliah saya. Alat ini sebagai kontrol ruang yang dapat dikontrol dari jarak jauh dengan menggunakan aplikasi telegram, sehingga memudahkan pengguna untuk memonitor ataupun mengontrol perangkat elektronik. Alat ini juga dilengkapi kontrol secara manual menggunakan push button, dan secara otomatis menggunakan sistem waktu dimana perangkat yang dikontrol akan on/off berdasarkan waktu yang di sudah diatur.

Cuci Piring Otomatis

Project ini merupakan tugas dari mata kuliah Instrumentasi Elektronika, yaitu pemanfaatan sensor sebagai kebutuhan sehari-hari, dimana saya menggunakan sensor IR (Infra Red) sebagai kontrol ON/OFF untuk mencuci perabotan dapur. Cara kerja alat ini yaitu apabila terdapat benda yang diarahkan ke sensor dengan jarak tertentu maka akan menyalakan kran air.

Hand Sanitizer Otomatis

Alat ini dibuat untuk perlengkapan pada balai desa piji, alat ini menggunakan sensor jarak (Ultrasonic).
